### Molar Mass Calculator
Find board when calculating the molar mass of a compound? This one may help. Go to MolarMassCalc/MolarMassCalc.py, download and then run this in terminal:
~~~ bash
$ python MolarMassCalc.py <compound_formula>
~~~

For example:
~~~ bash
$ python MolarMassCalc.py H2O
The molar mass of H2O is 18.02 g/mol
~~~
